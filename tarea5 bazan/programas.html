HTML

HTML , que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Sirve para definir la estructura y el contenido de una página web mediante el uso de etiquetas y elementos que permiten incluir texto, imágenes, enlaces, vídeos y otros recursos multimedia

CSS

CSS , que significa "Cascading Style Sheets" (Hojas de Estilo en Cascada), es un lenguaje de diseño que define la presentación visual de un documento HTML o XML, como una página web. En otras palabras, CSS controla elementos como el color, el tamaño, la fuente, el fondo, la alineación y el posicionamiento de los elementos dentro de la página web

JAVASCRIPT

JavaScript es un lenguaje de programación de alto nivel, interpretado, que se utiliza principalmente para hacer que las páginas web sean interactivas y dinámicas . Permite añadir funciones como actualizaciones de contenido, mapas interactivos, animación y otros efectos que mejoran la experiencia del usuario.

PHP

PHP PHP (acrónimo de "PHP: Hypertext Preprocessor") es un lenguaje de programación de código abierto, ampliamente utilizado en el desarrollo web. Permite crear sitios web dinámicos, donde el contenido puede cambiar según la interacción del usuario. Es especialmente útil para crear aplicaciones del lado del servidor, procesar datos y conectarse con bases de datos.
